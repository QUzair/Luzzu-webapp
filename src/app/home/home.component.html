• Luzzu - Data Quality Assessment Framework

  • Luzzu is a Quality Assessment Framework for Linked Open Datasets. It is a generic framework based on the Dataset Quality Ontology (daQ), allowing users to define their own quality metrics. We aim to provide an integrated platform that:
  • Assesses Linked Data quality using a library of generic and user-provided domain specific quality metrics in a scalable manner;
  • Provides queryable quality metadata on the assessed datasets;
  • Assembles detailed quality reports on assessed datasets.
